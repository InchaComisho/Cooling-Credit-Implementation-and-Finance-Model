# 保険・クーリングクレジット・リスク低減モデル

## 保険上の提案

暑熱、洪水、干ばつ、インフラのリスク低減を測定できれば、保険・再保険は事業をリスク低減投資として扱い得る。ただし全保険金が減る証明ではない。

期待年間損失（EAL）は事象確率×損失額の合計である。頻度、脆弱性、深刻度の変化に証拠がある範囲でのみ事業後EALを変更し、基準EAL－事業後EALから不確実性と残余リスクを差し引く。保険料には経費、資本費、モデル不確実性、利潤も含まれるため、EAL低下と保険料低下は一致しない。

## 経路

| 経路 | 測定 | 保険との関係 |
|---|---|---|
| 洪水ピーク | 貯留、浸透、流出係数、浸水深 | 財物・休業EAL |
| 熱中症 | WBGT曝露時間、発症、施設冷却 | 医療・労災 |
| 財物暑熱 | 表面・室温、設備ストレス | 財物・機械 |
| 農業 | 土壌水分、干ばつ損失、収量分散 | 農業・指数保険 |

センサーは気温、降雨、土壌水分、流出のパラメトリック保険トリガーを補完できる。ベーシスリスクを検証し、事業成果と保険支払トリガーを分離する。

## 支払合理性とデータ

割引後の期待保険金・資本変動の低下が、支払・検証・残余リスクを上回る場合、基準調査、保険料連動還元、強靱化助成、複数年成果支払が合理的である。価格ではなく独立検証した物理性能を条件とする。

位置付き曝露、災害履歴、損失、脆弱性曲線、資産・利用、気象・水文、匿名健康、保守、校正、対照、不確実性を要求する。逆選択、モラルハザード、モデル劣化、相関災害、保守不全、便益流出、保険料・クレジット・公的補償間の二重計上を監視する。これは保険数理予測・料率指示・保証ではなく、規制下の専門家承認が必要である。
