# 自治体クーリングクレジット実装ロードマップ

## 目的と成果ロジック

将来の暑熱、医療、エネルギー、災害負担の一部を予防資産へ振り替える。日陰・水循環型空間 → 暑熱曝露と流出の低下 → 医療費、冷房需要、災害負担の低下 → 検証済み成果への成果連動資金、という因果を検証する。

## 優先地点とベースライン

学校、病院、高齢者施設、駅前広場、公園、密集住宅地から始める。少なくとも一暖候期について、校正済みセンサー、電力請求、匿名化した熱中症記録、浸透試験、緑被率を取得し、対照地点とデータ管理規則を施工前に定める。

## 7段階

| 段階 | 実施内容 | 判断ゲート |
|---|---|---|
| 0：机上設計・基準準備 | 暑熱、脆弱性、流出、所有、費用を地図化 | 比較可能な境界を承認 |
| 1：小規模実証 | 優先地点で介入を実装 | 安全・調達・保守責任を確定 |
| 2：MRV・検証 | WBGT、表面温度、電力、熱中症、浸透、緑被を測定 | 独立検証者が方法と不確実性を承認 |
| 3：試行発行 | 追加的な検証成果に保守的な単位を付与 | 二重計上防止と番号管理 |
| 4：成果連動金融・官民資金 | 受益者と成果支払を契約 | 支払条件と損失分担を明記 |
| 5：登録・償却・制度化 | 登録、移転、償却、監査、開示 | 条例または制度規則を採択 |
| 6：分野拡大 | 農業、森林、沿岸、観光、乾燥地へ展開 | 地域証拠で移転可能性を確認 |

## 指標・資金・統治

WBGT、表面温度、気象補正後の冷房電力、プライバシーに配慮した熱中症件数、雨水浸透、貯留、緑被率・樹木生存を追跡する。自治体予算・国庫補助に、保険、電力、不動産、ESG投資を組み合わせる。前提、調達、利益相反、反転リスク、保守、苦情処理、償却を公開し、保守的感度分析でも便益が残る場合に限り拡大する。
