# خارطة طريق البلديات لتنفيذ أرصدة التبريد

## الهدف ومنطق النتائج

يمكن تحويل جزء من أعباء الحرارة والصحة والطاقة والكوارث المستقبلية إلى أصول وقائية. التسلسل السببي هو: الظل وإدارة المياه → خفض التعرض للحرارة والجريان → خفض الكلفة الطبية والطلب على التبريد وعبء الكوارث → نتائج متحققة قابلة لتمويل النتائج.

## المواقع وخط الأساس

تبدأ الأولوية بالمدارس والمستشفيات ودور المسنين وساحات المحطات والحدائق والأحياء الكثيفة. يجمع خط أساس لموسم حار واحد على الأقل باستخدام حساسات معايرة وفواتير الطاقة وسجلات صحية مجهولة واختبارات التسرب وخرائط الغطاء الأخضر، مع مواقع ضابطة وحوكمة بيانات محددة مسبقًا.

## مراحل التنفيذ

| المرحلة | العمل | بوابة القرار |
|---|---|---|
| 0: التصميم المكتبي وخط الأساس | رسم الحرارة والهشاشة والجريان والملكية والكلفة | اعتماد حدود قابلة للمقارنة |
| 1: مشروع صغير | تنفيذ حزمة محدودة | تثبيت السلامة والمشتريات والصيانة |
| 2: القياس والإبلاغ والتحقق | قياس WBGT والسطح والطاقة والصحة والتسرب والخضرة | قبول مستقل للمنهج وعدم اليقين |
| 3: إصدار تجريبي | وحدات محافظة لنتائج إضافية مثبتة | منع العد المزدوج وترقيم الوحدات |
| 4: تمويل قائم على النتائج | عقود دفع من المستفيدين | توثيق محفزات الدفع وتوزيع المخاطر |
| 5: السجل والإلغاء والمؤسسية | تسجيل ونقل وإلغاء وتدقيق | اعتماد قاعدة بلدية |
| 6: التوسع | الزراعة والغابات والسواحل والسياحة والأراضي الجافة | إثبات قابلية النقل محليًا |

## المؤشرات والتمويل والحوكمة

تتابع WBGT وحرارة السطح والكهرباء المصححة مناخيًا وحالات ضربة الحرارة والتسرب والتخزين والغطاء الأخضر. يمزج التمويل البلدي والدعم الوطني مع شركات التأمين والمرافق والمطورين ومستثمري ESG. تنشر الافتراضات والمشتريات وتعارض المصالح والصيانة والتظلمات وإلغاء الوحدات، ولا يتم التوسع إلا إذا بقيت المنافع جوهرية في السيناريو المحافظ.
