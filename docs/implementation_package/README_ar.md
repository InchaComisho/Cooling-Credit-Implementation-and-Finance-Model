# حزمة تنفيذ ومحاكاة أرصدة التبريد

تحول هذه الحزمة مفهوم أرصدة التبريد إلى مواد عملية للبلديات والحكومات وشركات التأمين والمجتمعات ومقدمي رأس المال.

> أرصدة التبريد ليست تبرعات، بل استثمارات مدفوعة مقدمًا لخفض المخاطر.

تقارن الحزمة تكاليف التنفيذ والتحقق بالخسائر المتجنبة في الصحة والطاقة والكوارث والتأمين والزراعة والمياه والاقتصاد الإقليمي. النماذج أدوات مفاهيمية لدعم القرار وليست تنبؤات أو ضمانات، ويجب استبدال افتراضاتها ببيانات محلية.

## تسلسل القرار

يبدأ العمل بخط أساس، ثم تجربة محدودة، فتحقق مستقل من النتائج، وإصدار وحدات قابلة للدفاع عنها، وعقود دفع مقابل النتائج، ثم التوسع بعد المراجعة. يجب توحيد حدود المقارنة والفترة والخصم بين سيناريو عدم التدخل والتنفيذ.
