# 都市冷却パイロット事業設計

## 推奨する初回実証

暑熱曝露が高く、公共所有または明確な同意、信頼できるメーター、比較可能な対照地区を持つ学校、病院、高齢者施設、駅前広場を選ぶ。徒歩圏の介入境界を定め、脆弱な利用者のアクセスを守る。

## 測定と介入

可能なら一暖候期、最低でも代表的猛暑日を含む8～12週間の基準測定を行う。WBGT、気温・表面温度、湿度、通行量、電力、暑熱事案、降雨、流出・浸透、水使用、植生状態を記録する。

介入は、樹木、日除け、雨水回収、保水性舗装、屋上・壁面緑化、雨水・再生水ミスト、植栽床への有機物・腐植循環を地点条件に合わせて組み合わせる。水ストレスや衛生管理上不適切な場合は飲用水ミストを用いない。

## 工程

| 期間 | 作業 |
|---|---|
| 0～3か月 | 統治、対象・対照選定、安全・測定規約 |
| 4～9か月 | 基準測定、詳細設計 |
| 10～15か月 | 調達、施工、試運転 |
| 16～27か月 | 暖候期を含む事後測定 |
| 28～30か月 | 独立検証、経済評価 |
| 31～36か月 | 試行発行、成果支払、拡大・停止判断 |

## MRV・費用・便益

WBGT、表面温度、ピーク冷房kWh/kW、熱中症・応急対応、浸透・貯留、緑被、生存率、水質、稼働率の対照差を追跡する。設計、許認可、土木、植栽、貯留、センサー、データ、MRV、検証、保守、更新、住民参加、予備費、撤去を費用化する。医療・電力回避、浸水低減、快適性、資産保護、保水、生物多様性、地域支出を便益として分離表示する。

## リスク管理

衛生、滑り、構造荷重、枯死、渇水、センサー偏り、個人情報、立退き、二重計上、過大主張を管理する。算式を事前登録し、保守的帰属と不確実性幅を示し、事故台帳と保守準備金を設け、安全性・データ充足・追加性・保守的純便益の基準を満たす場合のみ進む。
