# 政府向け意思決定ブリーフ：先払い型リスク低減としてのクーリングクレジット

## 判断事項

独立評価を伴う限定的自治体実証と省庁横断ベースラインを承認する。広範なクレジット義務化から始めない。

政府は猛暑、洪水、干ばつ、救急医療、インフラ損傷、農業損失の後に既に支払っている。クーリングクレジットは負担の一部を事後復旧から被害前のリスク低減へ移す。

## 公共価値

- **財政:** 無対策支出と、資本・運営・MRV・残余損失を同じ割引条件で比較する。
- **医療:** 日陰動線と公共施設冷却は曝露・熱中症負担を下げ得る。健康データは匿名化する。
- **災害復旧:** 浸透、貯留、植生、土壌回復は流出ピークと期待損失を下げ得るが、排水基準を代替しない。
- **公共施設:** 気象補正メーターで冷房需要・ピークを検証する。
- **水循環:** 保水は流出・灌漑需要を抑え得るが、水質・渇水制約を守る。
- **農業・食料:** 土壌水分と投入効率は収量安定に寄与し得るが、地域作物データが必要である。
- **地域強靱性:** 地域雇用、保守、快適性、観光、資産保護を分離し二重計上を防ぐ。

## 公費投入の根拠

便益は複数部局へ分散し、通常予算より遅れて実現する公共財である。基準データ、公開MRV、脆弱層のアクセス、初期実証には公費の合理性がある。電力、保険、所有者、投資家には受益に応じた契約負担を求める。

## 最初の90日

主管課と保健・環境・水・財政のチームを設置し、対象・対照地点を一つ選び、20年共通評価様式、基準測定費、公開設計・独立検証調達を承認する。追加性、保守的基準、不確実性、保守準備金、透明調達、地域同意、二重発行防止、償却、停止・拡大ゲートを必須とし、モデルを予算予測ではなく選別資料として扱う。
