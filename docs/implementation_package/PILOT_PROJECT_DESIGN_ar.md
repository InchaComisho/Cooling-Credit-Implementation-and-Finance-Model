# تصميم مشروع تجريبي للتبريد الحضري

## التجربة الأولى

يختار موقع مدرسة أو مستشفى أو دار مسنين أو ساحة محطة عالي التعرض، بملكية أو موافقة واضحة وعدادات موثوقة ومنطقة ضابطة مماثلة. تحدد منطقة تدخل قابلة للمشي وتحفظ إمكانية وصول الفئات الهشة.

## القياس والتدخل

يجمع خط أساس لموسم حار، أو 8–12 أسبوعًا تشمل أيامًا حارة ممثلة. تسجل WBGT وحرارة الهواء والسطح والرطوبة والحركة والكهرباء والحوادث الحرارية والمطر والجريان والتسرب والمياه وحالة النبات.

تجمع الحزمة بين أشجار الظل والهياكل المظللة وحصاد المطر والرصف المحتفظ بالماء والأسطح والجدران الخضراء والرذاذ بمياه المطر أو المياه المعاد تدويرها ودورة المادة العضوية والدبال. لا يستخدم ماء الشرب للرذاذ عندما تكون ندرة المياه أو الصحة عائقًا.

## القياس والكلفة والمنافع والمخاطر

تقاس الفروق المقترنة في WBGT والسطح وذروة الكهرباء والحوادث الصحية والتسرب والتخزين والغطاء الأخضر وبقاء النباتات وجودة المياه والتشغيل. تشمل الكلفة التصميم والتراخيص والأعمال والنبات والتخزين والحساسات والبيانات والتحقق والصيانة والاستبدال والمشاركة والاحتياط والتفكيك. تشمل المنافع الصحة والطاقة والفيضانات والراحة وحماية الأصول والمياه والتنوع والإنفاق المحلي.

تدار مخاطر النظافة والانزلاق والحمولة وموت الأشجار والجفاف وانحراف الحساس والخصوصية والتهجير والعد المزدوج والمبالغة. تسجل الصيغ مسبقًا ويستخدم الإسناد المحافظ ونطاق عدم اليقين وسجل الحوادث واحتياطي الصيانة.
