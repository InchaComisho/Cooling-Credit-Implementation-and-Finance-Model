# クーリングクレジット投資家・SPVモデル

## 資金構造

```text
投資家
  ↓
クーリング事業SPV／基金
  ↓
冷却事業
  ↓
測定済み成果
  ↓
成果支払＋クレジット収入＋回避費用契約
  ↓
投資家還元＋地域還元＋再投資
```

SPVは契約、資産、データ権、準備金、責任を隔離するが、不確実な社会便益を保証利回りへ変換してはならない。

## 資金使途・収入・指標

CAPEX、OPEX、MRV、登録、保険、開発、更新、準備金を資金使途とする。自治体成果支払、保険リスク低減支払、電力ピーク削減契約、不動産価値捕捉、農業・観光便益分配、保守的に発行したクレジットを収入源とする。

回収年は累積投資家CF、ROIは無割引の生涯純利益÷初期投資、NPVは要求収益率で算定する。IRRは符号変化があり経済的に意味ある根が得られる場合のみ示す。社会的総便益と契約済み現金収入を混同しない。

## 感度・リスク開示

価格・数量、工事超過、遅延、資産生存、OPEX、検証不合格、成果支払減額、割引率、契約更新を基準・下方・厳格下方で試す。追加性、永続性、所有、表示、同意、調達、相手先信用、データ、税、金融規制、倒産隔離を精査する。

## 投機を先行させない理由

物理性能、MRV、登録、購入者表示、償却が流通性に先行する。初期投機は価格と成果を切り離し、二重計上と地域への変動転嫁を招く。回避費用を受ける主体との契約収入を第一とし、クレジットは補完に置く。

安全運転、MRV、登録、債務を優先し、準備金と開示済み地域還元を確保してから残余を投資家へ配分する。収益は保証せず、独立検証と年次CF監査を行う。
