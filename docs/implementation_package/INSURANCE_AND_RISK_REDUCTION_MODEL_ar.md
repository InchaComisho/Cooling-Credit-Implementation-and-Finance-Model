# نموذج التأمين وخفض المخاطر لأرصدة التبريد

## الفرضية التأمينية

إذا أمكن قياس خفض مخاطر الحر والفيضانات والجفاف والبنية، يمكن لشركات التأمين وإعادة التأمين اعتبار المشروع استثمارًا في خفض المخاطر، لا دليلًا على انخفاض كل المطالبات.

الخسارة السنوية المتوقعة EAL هي مجموع احتمال الحدث × شدته. تتغير فقط مع دليل على تغير التكرار أو الهشاشة أو الشدة. الخسارة المتجنبة هي EAL الأساسي ناقص EAL بعد المشروع مع عدم اليقين والخطر المتبقي. يشمل القسط المصروف ورأس المال وعدم اليقين والهامش، لذلك لا يساوي خفض القسط خفض EAL.

## المسارات

| المسار | القياس | الصلة |
|---|---|---|
| ذروة الفيضان | التخزين والتسرب والجريان والعمق | الممتلكات وتعطل الأعمال |
| ضربة الحرارة | ساعات WBGT والحوادث والتبريد | الصحة وتعويض العمال |
| حرارة الممتلكات | حرارة السطح والداخل وإجهاد المعدات | الممتلكات والأعطال |
| الزراعة | رطوبة التربة وخسارة الجفاف وتباين الغلة | تأمين المحاصيل والمؤشرات |

يمكن للحساسات دعم تأمين بارامتري يعتمد على الحرارة أو المطر أو الرطوبة أو الجريان. يجب اختبار مخاطر الأساس وفصل أداء المشروع عن محفز المطالبة.

## لماذا قد تدفع شركة التأمين

يمكن تمويل الدراسات أو الخصم المرتبط بالقسط أو منحة المرونة أو الدفع متعدد السنوات إذا تجاوز خفض المطالبات وتقلب رأس المال المخصوم كلفة الدفع والتحقق والخطر المتبقي. يرتبط الدفع بأداء مادي متحقق لا بسعر الرصيد.

تطلب بيانات مكانية للتعرض والخطر والخسارة والهشاشة والأصول والطقس والمياه والصحة المحمية والصيانة والمعايرة والضوابط وعدم اليقين. تراقب الانتقاء والمخاطر الأخلاقية وانحراف النموذج والكوارث المترابطة وفشل الصيانة والعد المزدوج. النموذج ليس توقعًا اكتواريًا أو تسعيرًا أو ضمانًا.
