# クーリングクレジットのガバナンス・グリーンウォッシュ防止モデル

本書は、クーリングクレジットが曖昧なESGラベル、投機的な証書、排出相殺の免罪符にならないためのガバナンス原則を定義する。

クーリングクレジットは、測定された物理的冷却成果と、自然冷却機能の回復に結びついていなければならない。

---

## 1. 基本原則

```text
測定された冷却がなければ、クーリングクレジットではない。
```

プロジェクトが環境に良さそう、人気がある、見た目が緑である、気候対策らしいという理由だけで、クーリングクレジットを発行してはならない。

発行できるのは、検証済みの冷却、または自然冷却機能の検証済み回復がある場合だけである。

---

## 2. ガバナンスの目的

ガバナンスは、次を守るために必要である。

1. クーリングクレジットという名称の意味
2. MRVの信頼性
3. カーボンオフセット主張との分離
4. 地域住民の権利と便益
5. 資金循環の健全性
6. プロジェクトの生態系安全性
7. 公開主張の透明性
8. 冷却機能の長期維持

---

## 3. 主なグリーンウォッシュリスク

| リスク | 内容 | 管理策 |
|---|---|---|
| ラベル膨張 | 何でもクーリングクレジットと呼ぶ | 境界確認とMRVを必須化 |
| 紙だけの冷却 | 物理データなしに証書を発行 | 測定指標を必須化 |
| オフセット誤用 | 排出継続の口実にする | 自動的な排出相殺主張を禁止 |
| 過大発行 | 弱い効果や一時的効果を過大評価 | 不確実性・持続期間で割引 |
| 二重計上 | 同じ成果を複数主体が主張 | レジストリと主張追跡 |
| 地域収奪 | 外部購入者が価値だけを取る | 地域便益の開示を要求 |
| 生態系被害 | 冷却事業が生態系を壊す | 生態系リスク審査 |
| 投機化 | 冷却維持と無関係に証書が売買される | 有効性を維持管理・MRVと連動 |

---

## 4. 必要な主張表現

クーリングクレジットの主張は、正確でなければならない。

望ましい表現：

```text
このプロジェクトは、明示された基準線、境界、監視期間、MRV方法の下で、検証済みの冷却貢献を生んだ。
```

避けるべき表現：

```text
このプロジェクトは当社の排出を相殺する。
このプロジェクトにより当社はカーボンニュートラルである。
このプロジェクトは気候変動を解決する。
このプロジェクトは永久的な冷却を保証する。
```

別制度で認証されていない限り、クーリングクレジットをカーボンオフセットとして表示してはならない。

---

## 5. 最低限の開示事項

発行されるクーリングクレジットまたは成果主張には、次を開示するべきである。

1. プロジェクト所在地
2. プロジェクト運営者
3. 購入者または支払者
4. 介入の種類
5. 基準線条件
6. 測定境界
7. 監視期間
8. 測定指標
9. 不確実性の範囲
10. リスクと副作用評価
11. 第三者検証の有無
12. 有効期間
13. 排出相殺ではなく冷却貢献であること

---

## 6. レジストリと主張追跡

レジストリには、次を記録する。

- プロジェクトID
- クレジット発行日
- 監視期間
- 購入者または主張主体
- 数量またはスコア
- 使用方法論
- 失効日
- 償却・使用状況
- 公開主張の制限
- MRV報告書へのリンク

これにより、二重計上を防ぎ、公開主張を追跡可能にする。

---

## 7. プロジェクト統治の役割

| 役割 | 責任 |
|---|---|
| プロジェクト運営者 | 冷却プロジェクトの実装・維持 |
| MRV提供者 | 成果の測定・報告 |
| 検証者 | データ、基準線、方法論の確認 |
| レジストリ管理者 | 発行、主張、償却、有効性の追跡 |
| 地方自治体 | 公共利益と地域適合性の確認 |
| 地域代表 | 地域便益と懸念の確認 |
| 購入者・支払者 | 検証済み成果へ資金を出し、主張を適切に使う |
| ガバナンス委員会 | 紛争、リスク、方法論更新の審査 |

---

## 8. 停止条件

次の場合、クレジット発行を一時停止または停止するべきである。

- MRVデータが欠落している
- センサー故障が放置されている
- 維持管理が停止している
- 冷却効果を介入に帰属できない
- 生態系被害が確認された
- 水ストレスが悪化した
- 主張が誤解を招く
- 二重計上が発生した
- プロジェクト運営者が報告義務に違反した

---

## 9. 地域コミュニティ保護

冷却プロジェクトは、公共空間、農地、森林、流域、沿岸、共有生態系で実施されることが多い。

ガバナンスでは、次を求めるべきである。

- 地域説明
- 透明な便益配分
- 地域からの苦情窓口
- 土地・水資源の囲い込み防止
- 既存の生業への配慮
- 意図しない負担の監視
- 要約された成果への公開アクセス

クーリングクレジットは、地域生態系から価値だけを取り出し、維持負担を地域へ残す制度であってはならない。

---

## 10. 金融上の健全性

金融ガバナンスでは、次を求めるべきである。

- プロジェクト資金と運営資金の分離
- 明確な収益配分
- 維持管理準備金
- MRV予算
- 監査記録
- 利益相反の開示
- 投機的主張の制限
- 各地域の金融法・証券法への適合

クーリングクレジットは、まず成果連動型金融と検証済み貢献会計として始めるべきであり、未規制の投機商品として始めるべきではない。

## 12. 要約

クーリングクレジットは、誤用から守られて初めて信頼性を持つ。

ルールは単純である。

```text
冷却主張 = 測定された冷却 + 透明なMRV + 責任あるガバナンス
```

これらがなければ、クーリングクレジットは、地球を冷やすはずの制度ではなく、地球システムから切り離された紙の証書になる危険がある。
