# 局所効果と全球効果の違い

現行グラフが控えめに見えるのは、境界を限定した直接経済を測るためである。

1. 単一都市事業は地域気候を変えない。
2. 単一農地は公的・供給網支援なしに回収できない場合がある。
3. 単一乾燥地ミストノードは降雨を生まない。
4. 単一沿岸事業はサイクロン強度を下げない。
5. 数千の冷却、保水、土壌、森林、乾燥地、都市、沿岸、海洋事業は背景リスク条件を変える可能性があるが、検証が必要である。

| 規模 | 示せること | 示せないこと |
|---|---|---|
| 局所実証 | 気温、WBGT、流出、費用、回収 | 大陸大気フィードバック |
| 自治体事業 | 都市暑熱、健康、洪水負担 | 全球水循環安定 |
| 地域流域 | 洪水ピーク、土壌水分、農業 | 全球気候緩和 |
| 大陸乾燥地ネットワーク | 熱勾配緩和・蒸発散回復仮説 | 降雨保証 |
| 全球実装 | 分散冷却・水循環安定化の可能性 | 確定的天候制御 |

## 二つの補完評価

> 局所シミュレーションは直接事業経済を測る。全球安定化シナリオはシステム回避リスクを測る。両方が必要である。

局所MRVは曖昧な惑星主張を防ぎ、停止・拡大判断を支える。地域・全球モデルは非線形相互作用、転移、遠隔影響、不確実性を試す。防御可能な帰属と支払契約がなければ、システム便益を局所クレジット収入に算入しない。

概念リスク圧力モデルは比較の物差しで、図の結果が実現する証拠ではない。
