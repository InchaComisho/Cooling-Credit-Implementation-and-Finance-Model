# クーリングクレジットの支払主体モデル

本書は、**誰がクーリングクレジットに支払うのか、なぜ支払うのか、支払うことで何を受け取るべきか**を整理する文書である。

クーリングクレジットは、善意だけに依存する制度では長続きしない。測定可能な冷却成果に対して、支払う側に合理的な理由があるとき、制度として持続しやすくなる。

---

## 1. 基本原則

基本的な資金原則は次である。

```text
熱被害によって損失を受ける主体が、
損失が発生する前に、検証済みの冷却成果へ資金を出せるようにする。
```

したがって、クーリングクレジット金融は次の性質を持つ。

- 気候リスク低減金融
- 熱被害予防金融
- 公衆衛生予防金融
- 防災投資
- インフラ強靭化投資
- 自然冷却機能回復投資

---

## 2. 主な支払主体

| 支払主体 | 支払う理由 | 期待される成果 |
|---|---|---|
| 自治体 | 熱中症、災害復旧、インフラ損傷、公共空間劣化 | 熱リスク低下、公的費用削減、安全な都市空間 |
| 国 | 気候適応、防災、食料・水資源安全保障 | 国家リスク低減、長期的な公費削減 |
| 企業 | ESG、労働安全、サプライチェーン安定、地域操業リスク | 検証済み冷却貢献とリスク低減 |
| 不動産開発者 | ヒートアイランド対策、資産価値、入居者快適性 | 涼しい地区と資産価値向上 |
| 電力会社 | ピーク電力低減、送電網安定 | 冷房需要低下と負荷平準化 |
| 保険会社 | 熱波、洪水、干ばつ、インフラ被害の保険金低減 | 保険損失の低下 |
| 農業・食品企業 | 土壌水分、収量安定、干ばつリスク | 生産地域の安定化 |
| 観光事業者 | 屋外快適性、景観回復、熱リスク低減 | 観光価値の維持・回復 |
| 漁業・沿岸地域 | 海洋熱、酸欠、沿岸生態系リスク | 沿岸レジリエンスと生態系状態の改善 |
| 投資家 | 成果連動型気候投資 | 測定可能な社会・環境・金融リターン |
| 地域住民 | 公衆衛生、快適性、生業、防災 | 地域冷却と地域還元 |

## 3. 支払ロジック

### 3.1 自治体

自治体はすでに、熱中症対策、クーリングシェルター、災害復旧、インフラ修繕、洪水対策、干ばつ対策、緊急支援に費用を支払っている。

クーリングクレジットは、これらの事後支出の一部を予防投資へ転換する仕組みである。

```text
被害後に払う → 被害前に払って被害を減らす
```

### 3.2 企業

企業は、操業リスク低減、労働者保護、サプライチェーン安定、地域評価向上、ESG・サステナビリティ報告のために参加できる。

ただし、支払いが排出継続の免罪符になってはならない。クーリングクレジットは、排出相殺ではなく冷却貢献として報告するべきである。

### 3.3 保険会社

保険会社は、熱波、洪水、干ばつ、農作物被害、不動産損傷、インフラ故障による保険金支払いを減らす直接的な関心を持つ。

気候関連損失の低下が検証されれば、保険料連動型または成果連動型の支払いが成立し得る。

### 3.4 電力会社

電力会社は、冷房ピーク需要の低下、送電網の安定、非常時予備力の軽減、地域過負荷リスクの低下につながる場合に、クーリングクレジットを支援できる。

### 3.5 投資家

投資家は、プロジェクトファンド、自治体債、成果連動型金融、ブレンデッドファイナンス、インフラ投資を通じて参加できる。

投資家リターンは、投機的な証書売買ではなく、検証済み成果に連動させるべきである。

## 4. 支払いメカニズム

| メカニズム | 内容 |
|---|---|
| 直接購入 | 購入者が検証済みクーリングクレジットを購入する |
| 成果連動契約 | 測定された冷却成果が達成された場合に支払う |
| 自治体冷却契約 | 自治体が地域冷却性能に対して支払う |
| 保険リスク低減支払い | 保険会社が検証済み損失低減リスクに対して支払う |
| 電力需要低減支払い | 電力会社がピーク冷房需要低減に対して支払う |
| 官民連携 | 公的主体と民間主体が冷却インフラを共同で支える |
| クーリングボンド | 債券資金で冷却プロジェクトを実施し、MRV報告と連動する |
| ブレンデッドファイナンス | 公的資金で民間投資リスクを下げる |
| 地域還元基金 | 収益の一部を地域保守・地域還元に戻す |

---

## 5. 購入者分類

クーリングクレジットの購入者は、三つに分けられる。

### 5.1 損失回避型購入者

熱・気候被害が直接的な損失になる主体である。

例：

- 自治体
- 保険会社
- 電力会社
- 農業・食品企業
- 観光事業者
- 不動産所有者

### 5.2 責任・開示型購入者

気候適応、都市冷却、公衆衛生、生態系回復への検証済み貢献を示すために支払う主体である。

例：

- 企業
- ESGファンド
- 大学
- 公的機関
- 大規模イベント運営者

### 5.3 戦略インフラ型購入者

冷却インフラが長期的な地域安定を守るために支払う主体である。

例：

- 国
- 開発銀行
- インフラファンド
- 気候基金
- 国際機関

---

## 6. 購入者が受け取るべきもの

購入者は、次を受け取るべきである。

1. プロジェクト説明
2. ベースライン説明
3. 測定された冷却指標
4. MRV報告書
5. リスクと不確実性の開示
6. 有効期間
7. 地域便益の説明
8. 別制度で認証されない限り、排出相殺ではないという明記

---

## 7. 購入者が受け取るべきではないもの

クーリングクレジットは、次を与えるものではない。

- CO₂排出許可
- 自動的なネットゼロ主張
- データのない曖昧なESG主張
- 公共的な自然機能の所有権
- 物理的冷却から切り離された投機的権利
- 一時的冷却成果に対する永久的主張

---

## 8. 要約

クーリングクレジット金融は、支払いが測定可能な冷却成果と合理的な損失低減に結びつくとき、信頼性を持つ。

中心となる問いは次である。

```text
熱の低下、災害リスク低下、冷房需要低下、自然冷却機能回復によって、誰が便益を受けるのか？
```

その便益を受ける主体こそが、クーリングクレジットの合理的な支払主体である。
