# كيفية قراءة النتائج الضعيفة

لا يلزم أن يسترد كل مشروع تكلفته سريعًا. النتيجة الضعيفة معلومة للقرار وليست عيبًا يجب إخفاؤه.

## 1. النتيجة الضعيفة ليست فشلًا

قد تنتج عن رأس مال مرتفع، أو نقص تقييم الخسائر المتجنبة، أو انخفاض افتراضات الصحة والكوارث والتأمين، أو إغفال ذروة الكهرباء والفضاء العام والأمن الغذائي والمياه، أو قصر الأفق، أو انخفاض سعر الرصيد، أو غياب الدعم.

## 2. ما الذي تكشفه

تحدد المشاريع التي لا يمولها القطاع الخاص وحده، والحاجة للدعم العام، ومؤشرات MRV الإضافية، والمنافع غير المقيمة، والافتراضات الحاسمة.

## 3. لماذا قد يبقى التنفيذ ضروريًا

ليست الحالة الأساسية المستقبل الوحيد. إذا ارتفعت الحرارة والأمطار والكلفة الطبية والتبريد ومطالبات التأمين وخسائر الزراعة، تتراكم خسارة عدم التدخل. لذلك تقيم الأرصدة كخفض مخاطر مدفوع مقدمًا، لا كربح سريع.

السيناريوهات ليست تنبؤات؛ تحدد بيانات التجربة التوقف أو إعادة التصميم أو الدعم أو التوسع.
