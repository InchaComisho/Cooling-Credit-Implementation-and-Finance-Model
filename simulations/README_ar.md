## الاستخدام

تثبت `numpy` و`pandas` و`matplotlib`، ثم يشغل ملف Python داخل كل مجلد. ينشئ النموذج `outputs/` وCSV تفصيليًا ورسوم PNG وملخصًا.

هذا نموذج مفاهيمي لدعم القرار، وليس توقعًا دقيقًا. يجب استبدال الافتراضات ببيانات محلية مقاسة ومراجعة قبل قرارات السياسة أو التأمين أو الاستثمار. النتائج حساسة للإسناد والنمو والخصم وبقاء الأصول والعقود؛ توحد الحدود ويمنع جمع المنافع المتداخلة.
